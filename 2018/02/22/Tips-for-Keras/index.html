模型可视化1

keras.utils.vis_utils模块提供了画出Keras模型的函数（利用graphviz）

该函数将画出模型结构图，并保存成图片：

plot_model接收两个可选参数：

  • show_shapes：指定是否显示输出数据的形状，默认为False
  • show_layer_names:指定是否显示层名称,默认为True

我们也可以直接获取一个pydot.Graph对象，然后按照自己的需要配置它，例如，如果要在ipython中展示图片

【Tips】依赖 pydot-ng 和 graphviz，若出现错误，用命令行输入

pip install pydot-ng & brew install graphviz

Recurrent Dropout

每个Recurrent Layer在Keras中有两种dropout：

  • a float specifying the dropout rate for input units of the layer；
  • recurrent_dropout, specifying the dropout rate of the recurrent units.
